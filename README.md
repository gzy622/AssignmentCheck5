# 作业提交追踪工具 PSA

面向课堂教学场景的作业提交效率工具，支持移动端与桌面端。

## 功能

| 模块 | 功能 |
|-----|------|
| 极速交互 | 点击切换状态，长按评分，批量模式 |
| 可视化 | 紧凑模式/信息模式，顶部进度条 |
| 评分 | 同界面录入分数，短按/长按区分 |
| 时间 | 日期选择，历史趋势图表 |
| 数据 | localStorage 存储，JSON 导入导出 |
| 无障碍 | 动画开关，高对比度模式 |

## 操作

- **点击**：切换已交/未交状态
- **长按**：打开评分弹窗
- **批量模式**：点击快速连续切换

## 技术

- 纯前端，无服务器依赖
- localStorage 本地存储
- 响应式设计，适配所有设备

## 文件结构

```
psa.html          # 主应用（HTML + CSS + JS）
├── <head>        # 样式定义
│   └── CSS       # 约 900 行（布局、组件、主题）
├── <body>
│   ├── 顶部栏    # 进度条、视图切换、日期选择
│   ├── 主内容    # 学生卡片网格
│   ├── 底部栏    # 导入/导出、批量模式
│   └── 弹窗      # 评分弹窗、设置面板、统计面板
└── <script>      # 约 400 行
    ├── DataManager    # 数据存储与导入导出
    ├── AppState       # 状态管理
    ├── UI             # 渲染逻辑
    └── Events         # 事件处理
```

## 使用

浏览器打开 `psa.html` 即可使用。
