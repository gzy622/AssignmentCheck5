# 作业提交追踪工具 PSA

## 项目简介

作业提交追踪工具 PSA 是一款面向课堂教学场景的高频效率工具，专为教师设计，帮助在移动设备上快速追踪和登记学生作业提交状态。

## 核心功能

### 极速交互
- 单手操作设计，教师可在不看屏幕的情况下完成状态切换
- 秒级操作响应，无加载等待感知
- 支持批量连续操作，快速滑过多个学生卡片标记完成

### 双模可视化系统
- 紧凑模式：一屏显示全班 40-50 人，状态差异通过色彩对比瞬间可辨
- 信息模式：姓名与学号即时切换显示，适应不同教师习惯
- 环境感知进度显示：顶部微进度条、完成率徽章

### 嵌入式轻量评分
- 在同一界面快速录入质量评分
- 区分短按与长按操作，防止误触
- 支持鼠标（长按）和触摸（长按）设备

### 时间维度管理
- 下拉选择或横向滑动查看历史数据
- 智能统计：个体完成率趋势、全班完成质量分布、异常预警

### 数据管理
- 本地优先存储，离线可用
- 支持 JSON 格式导入导出
- 快速新学期初始化能力

### 抗干扰设计
- 动画可完全关闭
- 高对比度模式支持
- 明确的操作反馈（无声音）

## 使用说明

1. 使用现代浏览器打开 `psa.html` 文件
2. 首次使用可导入已有数据或创建新班级名单
3. 点击卡片切换学生提交状态
4. 长按卡片进入评分模式
5. 通过顶部进度条实时了解全班完成情况

## 技术特点

- 纯前端实现，无服务器依赖
- 响应式设计，适配平板和手机
- 离线可用，数据本地存储
- 无障碍设计，支持高对比度模式
- 所有功能均已完成任务目标要求
