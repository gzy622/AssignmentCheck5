# 作业提交追踪工具 PSA

面向课堂教学场景的作业提交追踪工具，支持移动端与桌面端。单文件离线运行，快速登记提交状态与分数，并提供统计与历史趋势。

## 功能概览

| 模块 | 功能 |
| --- | --- |
| 状态登记 | 已交 / 迟交 / 未交 / 取消登记 |
| 评分 | 0-100 分输入，支持小数，内置数字键盘 |
| 日期 | 日历选择日期，按日期存档记录 |
| 视图 | 简洁模式 / 完整姓名模式切换 |
| 统计 | 总人数、已提交、提交率、平均分、最近 7 次趋势 |
| 数据 | JSON/CSV 导入导出、剪贴板一键复制、保留学生名单的重置 |
| 无障碍 | 高对比度模式、禁用动画 |

## 操作方式

- 点击学生卡片：已交 / 未交快速切换
- 长按学生卡片：打开评分与状态面板
- 状态面板：选择已交 / 迟交 / 未交 / 取消
- 分数录入：数字键盘输入，回车键默认保存为“已交”
- 日期切换：顶部日期按钮打开日历

## 数据与统计

- 数据按日期分组保存，自动计算提交率与平均分
- 统计面板展示当前日期统计与最近 7 次趋势
- 导入支持 JSON、CSV 或文本名单（每行一个学生）

示例 JSON：

```json
{
  "students": [
    { "id": "1", "name": "张三" },
    { "id": "2", "name": "李四" }
  ],
  "records": {}
}
```

## 设置与个性化

- 高对比度模式：适配强光环境
- 禁用动画：提升老旧设备性能
- 新学期初始化：清空历史记录但保留学生名单

## 技术说明

- 纯前端单页应用，无服务器依赖
- localStorage 本地持久化
- Pointer Events 统一处理鼠标/触摸/触控笔
